# Phase 3: Visual Captioning & Table of Contents (ToC)

## Overview
In this phase, we ingest the JSON array of event triggers generated by the Phase 2 Sentry. For each timestamp identified as an "Interest Zone", we extract the corresponding visual frame from the raw Ego4D video. We then pass this frame through a fast, quantized VLM (Moondream2 via MLX) to generate a dense semantic caption of the user's action. The output is a highly structured Table of Contents (ToC).

## Optimization for Apple Silicon
To maintain speed without sacrificing capability, we use the `mlx-vlm` library to run Moondream2. The models should be cached on the 2TB SSD to avoid clogging the internal M4 Mac storage.

## Pseudocode Implementation

```python
import cv2
import json
import os
from mlx_vlm import load, generate

# System Configuration
SSD_BASE = "/Volumes/Extreme SSD/ego4d_data"
MODEL_CACHE = os.path.join(SSD_BASE, "models/moondream2")

class CaptioningEngine:
    def __init__(self):
        # Load Moondream2 using MLX optimizations
        print("Loading Moondream2 into unified memory...")
        os.environ['HF_HOME'] = MODEL_CACHE # Force HuggingFace to use SSD
        self.model, self.processor = load("vikhyatk/moondream2")
        self.prompt = "What action is the user currently performing with their hands? Be specific."

    def extract_frame(self, video_path: str, timestamp_s: float) -> str:
        """ 
        Extracts a specific frame and saves it temporarily to the SSD.
        Returns the path to the saved image.
        """
        cap = cv2.VideoCapture(video_path)
        
        # Seek to the requested timestamp (in milliseconds)
        cap.set(cv2.CAP_PROP_POS_MSEC, timestamp_s * 1000)
        ret, frame = cap.read()
        cap.release()
        
        if not ret:
            raise ValueError(f"Could not read frame at timestamp {timestamp_s}")
            
        temp_img_path = os.path.join(SSD_BASE, "cache/phase3/temp_frame.jpg")
        cv2.imwrite(temp_img_path, frame)
        return temp_img_path

    def process_video_toc(self, video_id: str):
        video_path = os.path.join(SSD_BASE, f"videos/{video_id}.mp4")
        trigger_path = os.path.join(SSD_BASE, f"cache/phase2/{video_id}_triggers.json")
        
        if not os.path.exists(trigger_path):
            raise FileNotFoundError(f"Triggers for {video_id} not found.")
            
        with open(trigger_path, 'r') as f:
            triggers = json.load(f)
            
        toc_entries = []
        os.makedirs(os.path.join(SSD_BASE, "cache/phase3"), exist_ok=True)
        
        print(f"Processing {len(triggers)} triggers for {video_id}...")
        
        for t in triggers:
            timestamp = t['timestamp']
            trigger_type = t['trigger_type']
            
            # 1. Grab visual context
            img_path = self.extract_frame(video_path, timestamp)
            
            # 2. Run Moondream inference
            caption_out = generate(
                self.model, 
                self.processor, 
                img_path, 
                self.prompt, 
                max_tokens=64,
                temperature=0.2
            )
            
            # 3. Construct ToC Entry
            entry = {
                "timestamp": timestamp,
                "trigger_type": trigger_type,
                "caption": caption_out.strip()
            }
            toc_entries.append(entry)
            print(f"[{timestamp}s | {trigger_type}]: {entry['caption']}")
            
        # 4. Save Final ToC
        toc_out_path = os.path.join(SSD_BASE, f"cache/phase3/{video_id}_toc.json")
        with open(toc_out_path, 'w') as f:
            json.dump(toc_entries, f, indent=4)
        
        print(f"ToC successfully constructed for {video_id}.")

if __name__ == '__main__':
    engine = CaptioningEngine()
    engine.process_video_toc("sample_ego4d_vid_001")
```

## Verification Strategy
- **Spot-Check Frame I/O:** Modify the `extract_frame` temp saving script to permanently save a selection of 10 triggers locally. Manually confirm the captured JPEG actually corresponds to an action event without cv2 timing mismatches.
- **Hallucination Rate:** Read through the generated ToC JSON. Check if `moondream2` is consistently returning generic phrases (e.g. "a person's hand") vs highly descriptive steps ("user is turning a wrench"). If so, adjust the text prompt parameters.
